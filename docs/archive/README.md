## Archive Policy

Documents are archived when:
1. They represent completed planning phases
2. They are superseded by newer documentation
3. They are point-in-time snapshots no longer current
4. They are session summaries now tracked in lab-notebook/

Archives are maintained for:
- Historical reference
- Understanding design evolution
- Audit trail of decisions
